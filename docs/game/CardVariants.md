Many things in Warframe have different variants

In this game many cards will also have variants.

Variants of cards are different cards.

Any number of variants can be in the same deck.

A variant of a card is a card that is an enhanced version of the original.

For example, a Kuva Lancer is a better version of the Grineer Lancer as it heals the Operator.
However the Kuva Lancer is of higher rarity and thus might not be worth it for some decks.
